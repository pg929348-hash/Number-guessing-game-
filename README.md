# 🎯 Number Guessing Game

A simple Python game where you try to guess a random number between 1 and 10. 
Built with love, patience, and 4 hours of debugging on mobile 😅

## 🚀 How to Play

1. Run the program
2. Enter a number between 1 to 10
3. You get 7 tries to guess the correct number
4. Game will tell you if your guess is too high or too low
5. Invalid input like letters won't count as a try

## 🕹️ Game Features

- **Input Validation**: Doesn't crash if you enter text like `abc` or `fh`
- **Hint System**: Tells you to choose higher or lower number
- **Try Counter**: You get max 7 attempts
- **Win/Lose Messages**: Celebrates when you win, reveals answer when you lose
